# Immidiate_Whatsapp_Chat
This is a simple application that can input raw phone number for directly chat into whatsapp. I mean, like if we use wa.me/[phonenumber]

Version 1
 - compact size window (230x150)
 - open at center of your display
 - ready to type the number without cursor click to input area
 - enter = submit (efficiently for work)
 - input area only accept number and backspace (not sure with delete)
